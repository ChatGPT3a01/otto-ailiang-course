三個新武器

寫大程式的整理術
📦

全域變數

宣告在最上面（函式外），整支程式都能用。適合「大家都要看的資料」，例如距離。

🔘

bool 開關

只有 true/false 兩種值。「有沒有障礙物」這種是非題就用它。

🧰

自訂函式

把一件事包起來、取個名字，例如 obstacleDetector()。loop 保持乾淨，隨叫隨用。

🏭 分工思維：loop() 當「指揮官」只看開關做決策；obstacleDetector() 當「偵察兵」專職量距離更新開關。各司其職＝專業程式。

逐段看懂：指揮官與偵察兵

分工後的程式多乾淨
// 指揮官：只看開關決定劇本
void loop() {
  if (obstacleDetected) { /* 閃避劇本 */ }
  else {
    Otto.walk(1, 1000, 1);
    obstacleDetector();      // 叫偵察兵去看
  }
}

// 偵察兵：專職量距離、更新開關
void obstacleDetector() {
  distance = Otto.getDistance();
  if (distance < 15) { obstacleDetected = true; }
  else { obstacleDetected = false; }
}
