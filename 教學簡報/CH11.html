for 迴圈解剖

括號裡三段，用分號隔開
for (int i = 0; i < 4; i++) {
  // 這裡面的事會重複 4 次
}
段寫法意思
① 起點int i = 0準備計數器 i，從 0 開始
② 條件i < 4只要 i 還小於 4，就繼續
③ 變化i++每做完一圈，i 加 1

i 的旅程：0 → 1 → 2 → 3 →（變 4，條件不成立）停。所以剛好 4 次。

逐段看懂：用 i 改變參數

迴圈不只重複，還能「每次不一樣」
for (int i = 0; i < 4; i++) {
  Otto.updown(1, 1400 - i * 200, 20);
  // i=0 → 1400-0   = 1400（慢）
  // i=1 → 1400-200 = 1200
  // i=2 → 1400-400 = 1000
  // i=3 → 1400-600 = 800（快！）
}

把 i 放進參數裡做算術，每一圈的動作就會漸變——漸快、漸慢、越走越遠……全靠這招。這是純手寫 8 行做不到的。
